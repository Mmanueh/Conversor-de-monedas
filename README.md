## Descripción
Este proyecto es un **Conversor de monedas** interactivo desarrollado en Java. El programa consume datos en tiempo real de la [ExchangeRate-API](https://www.exchangerate-api.com/), permitiendo realizar conversiones entre diversas monedas globales con tasas de cambio actualizadas al instante.

Fue desarrollado como parte del desafío de formación de **Alura Latam**, aplicando conceptos de Programación Orientada a Objetos, consumo de APIs REST y manejo de archivos JSON.
